## 📝 Descripción
**My Compound Interest** es una herramienta web interactiva diseñada para la planificación financiera y el análisis de ahorros a largo plazo. La aplicación permite calcular de forma dinámica el crecimiento de un capital inicial aplicando el efecto del interés compuesto, ofreciendo proyecciones claras según diferentes variables financieras.

### 🎯 Objetivos del Proyecto
* **Facilidad de Uso:** Permitir al usuario realizar simulaciones complejas de manera intuitiva.
* **Proyección Detallada:** Visualizar el impacto real de las aportaciones periódicas en combinación con las tasas de interés.
* **Optimización:** Garantizar una experiencia de usuario rápida y fluida sin cargas innecesarias.

---

## ✨ Características Principales
* **Cálculo de Interés Compuesto:** Simulación inmediata basada en el capital inicial, tasa anual y periodo de tiempo (años).
* **Aportaciones Periódicas:** Opción para configurar depósitos adicionales adicionales (mensuales o anuales).
* **Diseño Mobile-First:** Interfaz adaptada al 100% para dispositivos móviles, tablets y ordenadores.
* **Actualización en Tiempo Real:** Visualización instantánea de los resultados al cambiar los valores del formulario.

## 💻 Ejecución en Entorno Local

Sigue estos pasos para clonar el repositorio e iniciar la aplicación en tu máquina local:

### 📋 Requisitos Previos
* **Node.js** (Versión LTS recomendada)
* Gestor de paquetes **npm**

### 🔧 Pasos de Instalación

1. **Clonar el repositorio:**
git clone https://github.com/Frangersal/my-compound-interest.git

2. **Navegar a la carpeta del proyecto:**
cd my-compound-interest

3. **Instalar dependencias de Node:**
npm install

4. **Levantar el servidor de desarrollo:**
npm run dev

5. **Abrir la aplicación:**
Introduce en tu navegador la dirección indicada por la terminal (normalmente http://localhost:5173).
